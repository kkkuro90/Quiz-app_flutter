# Настройка OpenAI API для генерации вопросов

## Как добавить свой API-ключ OpenAI

1. **Получите API-ключ OpenAI:**
   - Перейдите на сайт: https://platform.openai.com/
   - Создайте аккаунт или войдите в существующий
   - Перейдите в раздел "API Keys" (https://platform.openai.com/api-keys)
   - Нажмите "Create new secret key"
   - Скопируйте созданный ключ (начинается с "sk-")

2. **Настройте API-ключ в приложении:**
   - Откройте файл `lib/config/api_config.dart`
   - Замените `'YOUR_OPENAI_API_KEY_HERE'` на ваш скопированный API-ключ
   - Пример:
     ```dart
     static const String openAIApiKey = 'sk-your-actual-api-key-here';
     ```

3. **ВАЖНО: Защитите свой API-ключ:**
   - Добавьте файл `lib/config/api_config.dart` в файл `.gitignore`, чтобы не выкладывать ключ в репозиторий:
     ```
     /lib/config/api_config.dart
     ```
   - Никогда не публикуйте API-ключи в открытом доступе

## Использование функции генерации вопросов

1. Запустите приложение
2. Перейдите в раздел создания квиза (учительская панель)
3. Нажмите кнопку "Загрузить вопросы из файла"
4. Выберите PDF или TXT файл с лекционными материалами
5. Приложение автоматически извлечет текст и сгенерирует вопросы с помощью AI
6. Сгенерированные вопросы появятся в списке вопросов квиза

## Поддерживаемые форматы файлов
- PDF (.pdf)
- Текстовые файлы (.txt)

## Ограничения и рекомендации
- Для университетского проекта OpenAI предоставляет $5 бесплатного кредита при регистрации
- Каждый вызов API тратит небольшое количество средств
- Рекомендуется использовать не слишком большие файлы (до 3000 символов), чтобы сэкономить средства
- Качество генерации зависит от содержания файла и структуры текста

## Устранение неполадок
- Если генерация не работает, проверьте правильность API-ключа
- Убедитесь, что подключение к интернету стабильно
- Проверьте формат файла - поддерживаются только PDF и TXT
- Если возникают ошибки, проверьте консоль на наличие сообщений об ошибках
