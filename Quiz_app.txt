"Quiz" — Мобильное приложение для интерактивных квизов в образовательном процессе.  

1. Введение  

1.1. Назначение документа  

Приложение предназначено для проведения интерактивных тестов и квизов среди учеников с использованием их смартфонов, как с участием учителя в реальном времени, так и в самостоятельном режиме.  

1.2. Цели проекта  

• Повысить вовлеченность учащихся в учебный процесс.  

• Предоставить учителям современный и удобный инструмент для проведения быстрых опросов и проверки знаний.  

• Автоматизировать процесс создания тестов за счет использования готовой базы вопросов. 

 • Сделать подготовку к урокам и проверку знаний более эффективной.  

1.3 Описание проекта Мобильное приложение, состоящее из двух основных модулей:  

• Режим Учителя: позволяет создавать и настраивать квизы, запускать их в режиме реального времени на уроке и просматривать статистику по ответам учеников.  

• Режим Ученика: позволяет присоединяться к активным квизам по коду или проходить сгенерированные тесты для самоподготовки. 2. Функциональные требования  

 

2.1. Модуль Учителя  

1) Авторизация  

• Регистрация и вход по email/паролю  

• Хранение сессии в памяти устройства  

2) Создание квизов  

• Ручное создание - интерфейс с кнопками и формами  

• Генерация из базы - интеграция с API вопросов  

• Типы вопросов: один ответ, несколько правильных вариантов ответа, ответ, который нужно написать самим  

Поддерживаемые форматы учебных материалов: 

Текстовые файлы (.txt) - прямое чтение 

PDF документы (.pdf) - извлечение текста через встроенный парсер 

Word документы (.docx) - конвертация в текст 

GIFT файлы (.gift) - импорт готовых тестов 

Процесс создания теста из файла: 

Загрузка файла через системный диалог выбора 

Автоматическое извлечение текстового содержимого 

Генерация вопросов с помощью AI на основе текста 

Предпросмотр и редактирование сгенерированных вопросов 

Сохранение теста в базу приложения 

3) Управление квизами 

Календарь для планирования квизов 

Настройка даты и времени проведения 

Просмотр истории проведенных квизов 

Экспорт тестов в формате GIFT для использования в других системах 

 

 

2.2. Модуль Ученика  

1) Участие в квизах  

• Ввод PIN-кода  

• Интерактивные кнопки ответов с изменением цвета при нажатии  

• Таймер обратного отсчета  

2) Самостоятельное обучение  

• Каталог тестов с фильтрацией по предметам и темам  

• Поиск и сортировка тестов • Сохранение прогресса в локальном хранилище  

2.3. Аналитика и отчетность  

1) Финансовая аналитика для учителя  

Статистика по успеваемости в разрезе периодов (день/неделя/месяц) 

Графики и диаграммы результатов  

Фильтрация по датам  

Сортировка результатов  

2.4. Системные функции  

1) Оффлайн-режим  

Кэширование созданных квизов  

Локальное сохранение результатов  

Синхронизация при появлении соединения 

 2) Конвертер оценок  

Перевод баллов в 5-балльную систему  

Настройка порогов оценок
