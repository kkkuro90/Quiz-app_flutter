# Обновленные инструкции по использованию функции генерации вопросов

## Текущая реализация

### Поддерживаемые форматы файлов:
- Текстовые файлы (.txt) - ПОЛНОСТЬЮ РАБОЧИЕ
- PDF файлы (.pdf) - ВРЕМЕННО НЕДОСТУПНЫ (требуют дополнительной настройки)

## Как использовать (текущая версия)

1. Убедитесь, что вы добавили ваш OpenAI API-ключ в `lib/config/api_config.dart`
2. Запустите приложение
3. Перейдите в раздел создания квиза (учительская панель)
4. Нажмите кнопку "Загрузить вопросы из файла"
5. Выберите TXT файл с лекционными материалами
6. Приложение автоматически извлечет текст и сгенерирует вопросы с помощью AI
7. Сгенерированные вопросы появятся в списке вопросов квиза

## Добавление поддержки PDF файлов (для будущего развития)

Если вы хотите добавить поддержку PDF файлов в будущем, выполните следующие шаги:

### Вариант 1: Использование стороннего пакета
1. Добавьте пакет для извлечения текста из PDF:
   ```bash
   flutter pub add path_provider
   flutter pub add ffi
   flutter pub add pdfx  # или другой пакет для работы с PDF
   ```

2. Обновите метод `extractTextFromPdf` в `lib/services/file_processing_service.dart`

### Вариант 2: Использование нативных возможностей
Для полноценной поддержки PDF текста может потребоваться:
- На Android: использовать PDFium библиотеку
- На iOS: использовать Core PDF framework
- Это требует нативного кода и более сложной настройки

### Вариант 3: Серверное решение
1. Загружайте PDF файл на сервер
2. Используйте Python/Node.js скрипт для извлечения текста
3. Отправляйте текст обратно в приложение

## Важные замечания

- В текущей реализации поддерживается только TXT файлы
- PDF поддержка была временно отключена из-за сложностей с извлечением текста
- Функция генерации вопросов с помощью OpenAI работает полноценно
- Для университетского проекта TXT файлов должно быть достаточно

## Устранение неполадок

- Если генерация не работает, проверьте правильность API-ключа
- Убедитесь, что подключение к интернету стабильно
- Проверьте формат файла - сейчас поддерживаются только TXT
- Если возникают ошибки, проверьте консоль на наличие сообщений об ошибках
